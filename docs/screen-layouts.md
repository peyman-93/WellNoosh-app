## Design Features

### Color Scheme
- Primary: Green (#10B981)
- Background: Gradient (light green to blue to purple)
- Cards: White with transparency
- Text: Dark gray (#1F2937) for primary, lighter for secondary

### Typography
- Headers: Bold, larger font
- Body text: Regular weight
- Buttons: Medium weight
- Small text: Lighter color

### Interactive Elements
- Rounded corners (8-16px)
- Subtle shadows
- Hover/press states
- Smooth transitions

### Layout
- Clean spacing (12-24px margins)
- Card-based design
- Consistent padding
- Responsive grid layouts

To see the actual rendered screens, you'll need to:
1. Run `npx expo start` 
2. Open in Expo Go app or simulator
3. Navigate through the screens
4. Take screenshots manually
